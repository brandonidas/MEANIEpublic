I developed this project from scratch, including self-learning web development, on my own. At the time, 2014-2015, I was a final year polytechnic student in Singapore, completing a Diploma of Financial Business Informatics - basically financial and business analytics.

In my coursework, I had written programs before but they were mostly for analyzing market or business data. Web development is mostly new to me, basic programming logic is there but I am fascinated by the new technologies in this area, and as much as this project was meant to help people make choices, it was also meant for me to learn about web development technology.

I did not build this project on established web technology, you will find no PHP, Ruby or Jquery here. I only wanted to learn upcoming technology. And as such, Choiciture was made only with the upcoming but not yet established. Namely, the Javascript full-stack abbreviated as "M.E.A.N".

M.E.A.N stands for
  • MongoDB
  • Express
  • Angular
  • Node

I only aimed my sights at upcoming technologies because I have national service starting in August and I wanted to jump ahead of the curve right now, even though learning the unestablished is harder than learning the established, so that when I come out of national service in 2017 I will have skills that are relevant, rather than obsolete.

Accompanying technologies include Git for version control, Bootstrap for mark up and many other open-source licensed projects (if any open-source contributor is reading this, thank you very much for enabling this to happen).

I would also like to thank Temasek Polytechnic's Entrepeneurship Center as well as Spring Singapore for granting the SPF Funding so that I could outsource certain aspects such as beta-testing and graphic design as well as buy learning resources including books and online courses.
